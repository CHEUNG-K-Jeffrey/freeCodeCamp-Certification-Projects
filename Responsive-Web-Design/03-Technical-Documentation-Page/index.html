Minimum requirements:

  • 4 GiB of Ram
  • 2 GHz Processor
  • 32 GB Hard Drive Storage

Recommended Requirements:

  • 8 GiB of Ram
  • 4 GHz Processor
  • 64 GB Hard Drive Storage
  • VR headset

Command-line Help
Picture by Tai Bui is licensed under Unsplash License.

To list command-line options:

$ cat-video-app --help

To run the app in the terminal:

$ cat-video-app

To run the app as a daemon:

$ cat-video-app --daemon

To run the app with GUI in the terminal:

$ cat-video-app --gui

To manually update the app:

$ cat-video-app --update
